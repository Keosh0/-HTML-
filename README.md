# Практическое задание: «Введение в HTML»

Данный репозиторий содержит результаты выполнения практического задания по дисциплине «Введение в HTML».

## Описание проекта
В рамках работы были созданы 4 HTML-страницы, демонстрирующие владение базовыми принципами веб-разметки:
1. `1 задание.html` — Базовая структура HTML-документа и форматирование текста.
2. `Задание 2.html` — Персональная страница (использование заголовков, списков, таблиц, гиперссылок и встраивание мультимедиа).
3. `Задание 3.html` — Форма обратной связи с базовой HTML5-валидацией (required, type="email", radio, checkbox).
4. `Задание 4c.html` — Страница с использованием семантических тегов HTML5 (header, nav, main, section, footer).

## Инструкция по запуску
Для просмотра страниц не требуется установка дополнительного программного обеспечения или локального сервера.
1. Склонируйте репозиторий или скачайте его архивом (Download ZIP).
2. Распакуйте архив в любую папку на вашем компьютере.
3. Откройте любой из файлов (`.html`) с помощью любого современного веб-браузера (Google Chrome, Safari, Яндекс.Браузер, Firefox).

## Структура файлов
- `index.html`, `about.html`, `form.html`, `semantic.html` — исходные коды страниц.
- `photo.jpg`, `audio.mp3`, `logo.png` — медиафайлы, подключенные к страницам через относительные пути.
